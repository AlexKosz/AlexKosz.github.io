Bug Tracker

A clean bug tracker app built using the MERN (MongoDb, Express, React, Node) stack and bootstrap. Users are required to log in/register to use the app. When signed in, users can view all bugs they have registered sorted first by priority, high to low, then by date, oldest to newest. Lastly, users can mark bugs as solved and provide their solution, solved bugs are then stored in a seperate tab with their solution
